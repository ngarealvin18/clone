About html

The p html element stands for a paragraph.

paragraphs are usually represented in visual media as blocks by blank lines and for the first-line indentation but html paragraphs can be any structural gr ouping of related content, such as images or form fields.

Paragraphs are block level elements and notably will automatically close if another block-level element is pass ed before the closing p tag.
